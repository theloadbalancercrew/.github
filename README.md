# The Load Balancer Crew

**Application delivery, network reliability, and infrastructure automation.**

The Load Balancer Crew helps teams plan, migrate, upgrade, troubleshoot, and automate the systems that sit between users and critical applications: load balancers, WAF, DNS/GTM, traffic policy, observability, and operational tooling.

F5 BIG-IP is a deep specialty for us, but our work is broader than one platform. We support enterprise teams, partner-led delivery, and infrastructure programs where change windows, rollback plans, and clear handoffs matter.

## What We Work On

- Application delivery and load balancing
- WAF tuning and traffic troubleshooting
- DNS/GTM, failover, and resiliency
- Infrastructure automation and runbooks
- MCP servers and AI-assisted operations tooling
- Technical staffing for infrastructure programs

## Open Source Direction

We are building practical tools for infrastructure operators: typed MCP servers, reviewable workflows, health-check helpers, and automation patterns that help engineers inspect systems safely before making changes.

The goal is not magic automation. The goal is better visibility, safer changes, and cleaner handoffs.
